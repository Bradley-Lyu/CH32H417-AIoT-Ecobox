# CH32H417-AIoT-Ecobox
An AIoT smart ecological box based on CH32H417 and CH584M
本项目为基于 **CH32H417** 与 **CH584M** 的 AIoT 智能生态箱控制系统，用于实现小型生态箱的环境监测、屏幕交互、无线通信和执行器控制。

## 项目简介

系统以 CH32H417 作为主控，负责环境控制逻辑、串口屏交互以及外设驱动；CH584M 负责蓝牙通信，实现移动端或无线端的数据交互。

本项目主要面向小型爬宠/生态饲养箱场景，目标是提高环境控制的自动化程度与交互便利性。

## 主要功能

- TJC 串口屏数据显示与交互
- CH32H417 主控程序
- CH584M 蓝牙通信
- 风扇、水泵、加热、喷雾、灯光等执行器控制
- 串口通信协议解析
- 环境状态显示与控制

## 硬件平台

- 主控 MCU：CH32H417
- 无线通信 MCU：CH584M
- 显示模块：TJC 串口屏
- 执行器：风扇、水泵、加热模块、喷雾模块、灯光模块等
